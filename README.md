# quickUbuntuBasedSetup
quick setup script

## How to use

if you really want to use it, just download it and:

```
chmod a+x qset-n.sh
sudo ./qset-n.sh
```

Quick Run:
(Copy & Paste)
```
git clone https://github.com/erentomurcuk/quickUbuntuBasedSetup.git &&
cd quickUbuntuBasedSetup
chmod a+x qset-n.sh &&
sudo ./qset-n.sh
```

! Do not sudo things that you don't know. !

## Optional Flags
- `-ure` installs `ubuntu-restricted extras` and MS Core Fonts
- `-rtc1` sets local time to RTC if you dual boot with Windows. (not really recommended but hey)
- `-rtc0` disables time being set to RTC and exits.
- `-h` or `-help` shows help.
